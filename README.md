Act as an expert front-end developer. Write a clean, semantic, and fully responsive HTML5 and CSS3 code for a modern Restaurant Website.

- Do not use any external CSS frameworks like Bootstrap—only pure CSS3.
- Ensure smooth scrolling and clean code comments.
